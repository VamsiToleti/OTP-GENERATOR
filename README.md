# 🔐 OTP Generator

A simple web app to generate a 4-digit OTP using JavaScript.

## 🚀 Features
- Generates a random 4-digit OTP
- One-click OTP copy button (add this if you implement it)

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript


## ▶️ Run Locally
1. Clone the repo
2. Open `index.html` in your browser
